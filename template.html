NOTES & TIPS
◆本手冊所有提示詞以 SDXL 架構為測試基準，DiT架構的模型及不同生圖平台的識別率可能有差異。
◆權重建議值為參考範圍，模型不同、配置不同都會影響效果。如果效果偏弱可嘗試調高 0.1–0.2，過度強調會導致畫面失真。
◆成衣型錄的「套裝提案」是本人嘗試過的不同紀錄，可作為靈感與參考。
◆手冊持續更新中，有新的詞條測試結果或組合配方會慢慢補充到對應區塊。

BASIC Q&A — 基本 QA
這個網站的 prompt 要怎麼用？▼
點擊任一卡片右下角的 COPY 按鈕，prompt 就會複製到剪貼簿。
直接貼到 PixAI 或其他算圖工具的正向 prompt 欄位即可。
建議在最前面放你自己的人設描述，再接服裝詞條。
（人設）是什麼意思？要怎麼填？▼
「人設」是指你的角色基本外觀描述，例如髮色、髮型、眼睛顏色、身材特徵等固定詞條，把這些詞統稱為「人設」。

把人設放在服裝 prompt 前面，模型才能在套上衣服的同時維持角色外觀。

範例：1girl, long black hair, blue eyes, maid dress, white apron
其中 1girl, long black hair, blue eyes 就是人設的部分。
經典衣櫃和成衣型錄有什麼差別？▼
· 經典衣櫃 — 收錄模型本來就認識的服裝詞組，例如護士裝就是從護士帽到鞋子的完整一套詞條，可以直接套用或拆開使用
· 成衣型錄 — 作者自己嘗試組合出來的穿搭詞條，是用製衣工坊的概念拼湊出的新組合

想找現成認識度高的服裝用經典衣櫃；想找有趣的穿搭組合靈感用成衣型錄。
製衣工坊的詞條要怎麼搭配使用？▼
製衣工坊把服裝拆成五個模組：版型、材質、剪裁、顏色、點綴。
建議的組合邏輯是：

版型（決定輪廓）→ 材質（決定質感）→ 剪裁細節（加特色）→ 顏色 → 點綴（可選）

散詞：dress, silk, off-shoulder, white, lace trim
組合成：white silk off-shoulder dress, lace trim，表示「白色絲質露肩洋裝，蕾絲裝飾」。

形容詞放在服裝詞前面，點綴類的配件詞用逗號接在後面即可。
可以參考頁面上的「組合示範」彈窗看更多例子。
寫真沙龍的「在 PixAI 開啟」連結是什麼？▼
點擊連結會直接在 PixAI 開啟對應的模型頁面，並帶入預設的 prompt 和參數設定。
你只需要換上自己的人設描述，其餘設定都已經調好，省去重新設定的步驟。
QUALITY — 圖面品質
圖片整體很模糊，細節不夠清晰▼
可以嘗試以下幾個方向：
① 增加 steps 或提高 cfg
② 在 prompt 加入 masterpiece, best quality, highres
③ 使用 Hires.fix 或圖生圖放大，效果比直接拉大尺寸好
圖片有很多噪點或顆粒感▼
通常是 steps 不夠或取樣方式設定的問題。
① 增加 steps
② 取樣方式改用 DPM++ 2M Karras 或 Euler a
③ 確認 denoise strength 沒有設太高（圖生圖模式）
圖面出現奇怪的色塊或色斑▼
色斑通常是 cfg 設太高導致模型過度渲染。
① 嘗試降低 cfg（PixAI 建議維持在 7 以下）
② 也可以換個 seed 重跑，有時候只是那張 seed 不穩定
人物比例看起來很奇怪，頭太大或手很詭異▼
手部變形是目前模型的通病，可以嘗試：
① 在正向 prompt 加入 correct anatomy, hand details
② 指定構圖讓手不出現在畫面內，例如 upper body、portrait
③ 用 inpaint 局部重繪手部區域
臉部崩掉或五官位置不對▼
① 在正向 prompt 加入 detailed face, perfect face
② 可以開啟 PixAI 的面部修復功能自動修正臉部
③ 也可以局部 inpaint 臉部區域重繪
想算人外角色（人魚、人馬、有義肢等），但一直長出不想要的部位▼
模型對人形有很強的預設傾向，遇到特殊體型時會自動「補全」成一般人體。建議：
① 明確描述你要的特殊部位，例如 mermaid tail, no legs, fish tail
② 在 negative prompt 排除不想要的部位，例如不想要腿就加 legs, human legs, feet
③ 越特殊的體型模型認識度越低，可能需要多跑幾次或調高相關詞條的權重
④ 如果有對應的 LoRA 會有效很多，建議搜尋看看有沒有相關資源
CLOTHING — 服裝生成
服裝出來是對的，但人物性別跑掉了（例如護士服變女生）▼
部分服裝詞條帶有性別刻板印象，模型會預設生成特定性別。如果要強調男性人設，建議在人設描述後面補充以下詞條：

male focus, manly, masculine, 1boy

例如：（你的人設）, nurse, male focus, manly, 1boy

反之若要強調女性，可以加 female focus, 1girl。詞條放在服裝詞附近效果較好。
男性人設穿女裝／女性人設穿男裝，體型特徵跑掉了▼
模型容易根據服裝「猜測」體型，導致穿女裝的男角胸部跑出來，或穿男裝的女角體型變中性。

可以在 negative prompt 裡明確排除不想要的特徵，例如：
· 男角穿女裝時加 female, breasts, wide hips 到 negative
· 女角穿男裝時加 male, flat chest 到 negative（依需求調整）

同時在正向 prompt 強調體型描述效果更好。
衣服根本沒出現，還是原本的便服▼
① 確認服裝詞條是否放在 prompt 前段（權重較高的位置）
② 嘗試用括號提高權重，例如 (maid dress:1.3)
③ 部分款式模型認識度較低，可以用更通用的詞替代，或參考經典衣櫃的建議詞條
兩種衣服的元素混在一起了▼
同時描述太多服裝類型時容易互相干擾。建議：
① 先確定主視覺的服裝，其他作為配件補充
② 使用製衣工坊的方式，從版型出發逐層疊加
③ 衝突的詞（如 dress 和 pants）盡量擇一描述
服裝顏色和我描述的不一樣▼
顏色詞要明確放在服裝詞前面，例如 red dress 而非 dress, red。
若要精確控色可以加 solid color 避免模型自由發揮，或提高顏色詞的權重。
服裝版型正確但布料材質感不對▼
可以直接在服裝詞前加材質，例如 silk dress、lace blouse、PVC skirt。
材質詞的效果因模型而異，可以參考製衣工坊的材質面料分類選用認識度較高的詞條。
LORA — LoRA 搭配
LoRA 是什麼？和模型有什麼不同？▼
底模（模型）決定整體的畫風基礎，可以把它想成畫圖用的紙和畫材。
LoRA 則是疊加在底模上的「風格或特徵補丁」，用來強化特定的畫風、角色特徵、或服裝效果，但不能單獨使用，必須搭配底模。

同一個 LoRA 套在不同底模上，呈現效果也會不同。
要去哪裡找 LoRA？▼
可以直接在 PixAI 平台上搜尋，有很多公開的 LoRA 資源。
搜尋時可以用功能描述下關鍵字，例如服裝風格、畫風名稱、角色名等。
建議選擇下載量或使用量較高、且有附範例圖的 LoRA，比較容易評估效果是否符合需求。
LoRA 的權重要設多少？▼
通常建議從 0.7–0.8 開始測試，觀察效果再調整。
· 權重太高（接近 1 或超過）容易讓畫面過於風格化，甚至出現變形
· 權重太低則效果不明顯

部分 LoRA 的作者會在說明頁建議適合的權重範圍，可以優先參考。
同時用多個 LoRA 會有問題嗎？▼
可以疊加使用，但需要注意：
① 多個 LoRA 的效果會互相影響，總權重建議不要太高，可以嘗試每個都稍微降低
② 風格差異太大的 LoRA 疊在一起容易打架，造成畫面混亂
③ 建議一次加一個測試，確認效果後再疊加第二個，比較容易排查問題
可以用服裝 LoRA 嗎？效果怎麼樣？▼
可以！服裝 LoRA 專門強化特定服裝的生成效果，對於模型原本認識度較低的款式特別有用。

使用方式和一般 LoRA 相同，建議搭配對應的觸發詞（trigger word）使用，通常在 LoRA 的說明頁會標註。
如果已經有服裝 LoRA，可以搭配本站的製衣工坊詞條一起使用，效果會更精準。
加了 LoRA 之後服裝出來了，但臉或身體跑掉了▼
服裝 LoRA 有時會連帶影響人物的臉部或體型，這是訓練資料的問題。
① 嘗試降低 LoRA 的權重
② 在正向 prompt 加強人設描述，明確指定你要的臉部和體型特徵
③ 如果影響太嚴重，可以考慮換一個同類型但訓練品質更好的 LoRA
LoRA 的觸發詞是什麼？一定要加嗎？▼
觸發詞（trigger word）是啟動 LoRA 特定效果的關鍵字，通常由 LoRA 作者設定。
有些 LoRA 沒有觸發詞也有效果，但有些必須加了觸發詞才會正常運作。

觸發詞通常標註在 LoRA 的說明頁，建議使用前先查看一下。
MODEL — 模型選擇
不知道要選哪個模型，有推薦嗎？▼
模型的風格差異很難用文字說清楚，而且套上不同 LoRA 後表現也會不一樣。
建議直接去經典衣櫃點開任一單品，可以看到五個模型用相同 prompt 的直出對比圖，實際效果一目瞭然。
同樣的 prompt 換模型後效果差很多▼
這是正常現象。不同底模對詞條的認識程度不一樣，建議：
① 先在多個模型各跑幾張觀察差異
② 經典衣櫃的對比圖可以幫助快速判斷哪個模型比較能識別該款服裝
③ 認識度低的模型可以加強詞條權重補救
同一個模型有很多版本，版本不同有差嗎？▼
有差，而且有時候差很多。以 Plant Milk 為例，不同版本在畫風細膩度、色調傾向和人物風格上都有明顯差異，同樣的 prompt 在不同版本上跑出來的感覺可能完全不同。

建議自己多試幾個找到喜歡的感覺。以 Plant Milk 為例，個人目前最常用的是 Plant Milk Almond，如果不知道從哪個版本開始可以先試這個。
要去哪裡選擇模型版本？▼
在 PixAI 的生圖介面裡，點擊模型名稱旁邊的版本號，就可以展開版本選單切換。

MISC — 其他問題
加了很多 prompt 但效果和沒加一樣▼
prompt 太長時後面的詞影響力會衰減。建議：
① 把最重要的詞放在最前面
② 刪除冗餘或重複的描述，聚焦在關鍵詞
③ 適當使用括號提高重要詞的權重，但不要全部都加，反而會互相抵消
每次生成的圖差異太大，風格不穩定▼
① 固定 seed 可以讓構圖和風格更穩定
② 降低 cfg 可以減少隨機性（建議 6–7 之間）
③ 增加 prompt 中對風格的描述，例如 anime style、illustration
這個網站的 prompt 可以直接複製去其他平台用嗎？▼
可以！這裡的 prompt 都是純文字服裝詞條，沒有綁定特定平台或模型。
不過不同平台的底模和參數設定不同，實際效果可能會有落差，建議搭配對應的基本品質詞使用。
